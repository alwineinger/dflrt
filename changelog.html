Changelog

Version 2.1

  • Added **Stages mode** for point-to-point relays, allowing configuration of stages instead of loops.
  • Stages mode includes numeric stage count entry, default distance option, or batch editing of names and distances.
  • Stage names default to numeric order, and default colors are removed while still allowing custom color selection.
  • Total Legs is automatically synced with the number of stages.
  • Leg column header changes to '#' and Loop column header changes to 'Stage' in Stages mode.
  • Switching between Loops and Stages triggers a full recalculation and resets actual paces.
  • Added **Pace** column in landscape mode showing actual or default pace for each leg.
  • Selective **Apply Changes** in Settings: updates only relevant data depending on what changed (avoids unnecessary recalculations).
  • Validation added when entering leg end times to ensure they occur after start times.
  • Improved unit handling: supports Imperial (miles/min per mile) and Metric (kilometers/min per km) modes.
  • Runner and Loop/Stage columns dynamically share space, prioritizing the longer value.
  • Support link added in Settings for quick access to help website.
